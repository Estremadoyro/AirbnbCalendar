## Aribnb-like calendar
This calendar playground aims to replicate **Airbnb's** calendar currently implemented on the app.

### Libraries / Frameworks
- Reactive: **Combine**
- CollectionViews: **NSDiffableDataSource & NSCompositionalLayout**
